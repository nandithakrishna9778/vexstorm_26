Process

The panel
in action.

Upload a resume and transcript. The agents take it from there ‚Äî no rubrics, no keyword lists, no shortcuts.

  1. 1
    Input Ingestion & Parsing

    Resume, transcript, and job description are tokenized and structured for each agent's specialized lens. No shared context between agents at this stage.

  2. 2
    Independent Parallel Analysis

    All four agents analyze simultaneously. Isolation prevents groupthink and anchoring bias that occurs if agents see each other's preliminary findings first.

  3. 3
    Cross-Examination Phase

    Agents share findings and formally challenge each other. The Gap Detector confronts the Interviewer with resume claims. Every challenge is logged.

  4. 4
    Weighted Voting

    Agents cast weighted votes based on confidence and evidence strength. Unanimous decisions are high-confidence. Contested decisions trigger additional scrutiny.

  5. 5
    Verdict & Reasoning Trace

    The Consensus Engine delivers a final Hire / Conditional / No-Hire verdict with a complete human-readable trace and specific follow-up actions required.
